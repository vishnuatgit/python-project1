# CLI Task Tracker

A pure Python command-line application that allows users to add, view, and delete tasks. 
The application teaches core Python concepts including:
- Dictionary manipulation (State management)
- File I/O (Reading and writing JSON files)
- Control Flow (`while` loops, `if/elif/else` statements)

## How to run
1. Ensure you have Python installed.
2. Open your terminal in this directory.
3. Run `python task_tracker.py`

This project serves as a foundational stepping stone toward understanding how AI Agentic Workflows manage state internally.
